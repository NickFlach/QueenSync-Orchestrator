You are Agent 4 working on QueenSync v1.0.

QueenSync is the agent operating layer for the Kannaka ecosystem. v0.1 and v0.2 are already being built. Your job is to work through the v1.0 arc in waves, preserving existing functionality while pushing toward a durable, deployable system.

Do not rewrite the app from scratch unless absolutely necessary. Inspect the current codebase first, understand existing modules, then evolve it.

Primary v1.0 goal:
Any agent can onboard, receive deterministic tasks, respond to Resonance Control Protocol fields, return outputs, pass through Memory Gate, and be observable from the Queen Console deployed at ninja-portal.com.
